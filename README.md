# ASTA: Women's Safety Intelligence Network 🛡️✨

ASTA is an AI-powered safety network designed to empower women with real-time risk intelligence, safest path routing, and community-driven incident awareness.

## 🚀 Key Features

- **AI-Safest Routing**: Dynamically calculates paths that prioritize safety over speed by injecting "Safe Hubs" into navigation.
- **Dynamic Risk Heatmaps**: Real-time visualization of urban safety levels powered by Spatio-Temporal Graph Neural Networks (ST-GNN).
- **Community Empowerment**: Horizontal horizontally scrolling community feed for safety articles, legal rights, and real-time incident reporting.
- **Cyberpunk UI/UX**: Premium, high-contrast dark theme with Glassmorphism and Neon accents for maximum readability and engagement.

## 🛠️ Technical Stack

- **Frontend**: React Native, React Navigation, Redux Toolkit, Google Maps API.
- **Backend**: AWS Serverless (Lambda, API Gateway), AWS SageMaker (ML Inference).
- **AI Engine**: ST-GNN (Spatio-Temporal GNN) for high-precision risk prediction.

## 📂 Project Structure

- `mobile/`: React Native mobile application.
- `backend/`: AWS Lambda functions and API logic.
- `ml/`: Python-based ML models and safety-scoring algorithms.
- `data/`: Urban safety datasets and processing scripts.
